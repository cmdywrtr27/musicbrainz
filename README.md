# musicbrainz
file naming code for musicbrainz picard

Code will produce a folder structure like this:

/Music
 - /Artist
    - /Album (Year)
       - Track number - Artist - Song Title.mp3
       
Example:

/Music
 - /Bruno Mars
   - /Doo Wops & Hooligans (2010)
     - /02 - Bruno Mars - Just The Way You Are.mp3 (complete filename)
